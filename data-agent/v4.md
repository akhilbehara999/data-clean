### 1. đź§  Cross-Session Project Memory (`project_memory.py`)
*   **Behavior**: Remembers structural patterns, schemas, statistical distributions, and previous cleaning actions across files and sessions.

### 2. đź“… Recurring Report Automation (`report_scheduler.py`)
*   **Behavior**: Detects repeated analysis commands (like running `eda`, `insights`, or `pivot`) across sessions on similar files and prompts you to save the sequence as a reusable template.
*   **Storage**: Saved in `report_templates.json`.
*   **Management**: Run `/templates` in v4 to see, configure, or execute automation plans.

### 3. đźš¨ Anomaly Watchdog Mode (`watchdog.py`)
*   **Behavior**: Monitors a database table, Google Sheet, or local file for changes. It takes a baseline metric snapshot and alerts you on statistical drift.
*   **Flags**:
    *   Row count changes (> 20%).
    *   Data drift (metric mean shift beyond 2.0 Z-score standard deviations).
    *   Data quality regressions (increase in null value counts).
    *   New categorical values (options not seen in baseline).
*   **Execution**:
    ```bash
    python main.py --watch sample_data.csv --interval daily
    ```

### 4. đź”Ś Live Data Connectors (`v4/data_connectors/`)
Allows you to pull data directly into the cleaning and agent pipeline from remote endpoints:
*   **Google Sheets**: `python main.py --connect sheets --id <sheet_id>` (requires `gspread` and `google_credentials.json`).
*   **Database (Postgres/MySQL)**: `python main.py --connect db --config db_config.json` (requires database drivers and config json).
*   **REST API**: `python main.py --connect api --url <url>` (requires `requests`).
*   **Gmail Attachments**: `python main.py --connect gmail --query "<query>"` (requires desktop token flow).

### 5. đźŽŻ What-If Simulation Engine (`simulation_engine.py`)

*   **Execution**: Inside the v4 console, type:

    *   `/simulate what if we removed segment where category is "Promo"`

### 6. đź”Ť Domain Specialists
*   **Behavior**: Run focused rules and flags tailored to business domains:
    *   `/specialist fraud`: Flags margin structuring, suspicious transactions, or statistical outliers.

    *   `/specialist inventory`: Checks stock turnover rates, safety thresholds, and supply chain flags.
